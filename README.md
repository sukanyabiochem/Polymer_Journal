# Polymer_Journal
This repository contains a coarsed-grained (CG) molecular dynamics workflow built around VMD, Tk/tcl, GROMACS, bash script. The pipeline performs four major tasks:


Run GROMACS MD simulation stages for minimization, NVT equilibration, NPT equilibration, and production dynamics in NPT condition.
Before the analysis in GROMACS production trajectory is processed and then analysis performed using GROMACS tools, Tk/tcl tools and Python.
After the analysis, the out files plotted using Python.
